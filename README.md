# 🛒 Amazon Seller Scraper Germany

Scrape Amazon.de sellers by category and get their **business contact details**
from their legally required Impressum (§5 TMG).

## What you get

| Field | Description |
|---|---|
| Company Name | Official business name |
| Email | From Impressum |
| Phone | From Impressum |
| Address | Full business address |
| VAT ID | USt-ID if available |
| Category | Amazon product category |
| Store URL | Direct link to Amazon seller profile |

## Input

- **Category** — Choose from 15 Amazon.de categories (Baumarkt, Elektronik, …)
- **Max Sellers** — How many sellers to scrape (1–500)

## Output

- 📊 **Excel file (.xlsx)** — Downloadable from Key-Value Store
- 📋 **JSON Dataset** — Accessible via Apify API

## Pricing

**Pay per seller** — you only pay for successfully scraped records.
1 seller = 1 token ($0.01). No hidden fees.

## Legal

All data is extracted from publicly available Impressum pages.
German law (§5 TMG) requires commercial sellers to publish this information.
